Epic2Castor Data Mapping Application

Interactive Shiny application for mapping EPIC hospital data to Castor EDC

Key Features:

  • Excel-like spreadsheet interface for data mapping with tab-based organization
  • Copy/paste functionality with automatic handling of related checkbox & radiobutton data
  • Auto-fill intelligence using medical dictionary, DeepL / MyMemory translation APIs, fuzzy matching, and optional ML predictions
  • Import Wizard with multi-format support (CSV, Excel, TSV), sheet selection, and reusable mapping templates
  • Study Dashboard with patient inclusion curves, per-form data completeness, and biobank sample overview
  • ETL pipelines for baseline, follow-up, biobank, and lab data with real-time progress monitoring
  • Direct Castor API integration for metadata refresh and batch data upload
  • SharePoint integration for remote file access and reload
  • Medical Dictionary Manager for maintaining English–Dutch term translations
  • Configurable keyboard shortcuts with performance monitoring panel
