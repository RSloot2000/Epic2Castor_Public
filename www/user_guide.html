Study Dashboard

The Dashboard provides a live overview of your Castor study data retrieved via the API. Data is cached locally for 30 minutes.

Patient inclusion Cumulative inclusion curve over time with site breakdown
Data completeness Per-form completion percentages and missing field counts
Biobank overview Sample collection status and biobank-specific metrics
Refresh Click the Dashboard button in the menu bar to open; data auto-refreshes when cache expires

Import Wizard

The Import Wizard provides a step-by-step interface for importing external data files with automatic structure detection, interactive column mapping, and data transformation.

Step 1: Select Type Choose import type (EPIC Baseline/Follow-up, Biobank, Lab Data, Custom)
Step 2: Upload File Select CSV/Excel file; auto-detects encoding, structure, and sheets
Step 3: Map Columns Match required columns to your file’s columns with validation
Step 4: Review & Export Preview transformed data and export to CSV
Key Features
  • Multi-format support: CSV, Excel (.xlsx/.xls), TSV with automatic detection
  • Sheet selection: Choose specific sheets from Excel workbooks
  • Template system: Save and reuse mapping configurations
  • Smart validation: Real-time feedback on required columns and data types
  • Sample preview: View actual data while mapping columns
  • Direct export: Transform and export to CSV in one workflow

SharePoint Integration

When the application detects it is running inside a SharePoint-synced folder, additional options become available.

File → SharePoint reload Reload input files from the SharePoint-synced location
Automatic redirection Input data paths are transparently redirected to the SharePoint temp folder when active

Tips
  • Use Ctrl+Shift+P to monitor performance during large operations
  • Tab switches preserve your active tab when switching between tables
  • Copy/paste operations automatically handle related checkbox and radiobutton data
  • The performance panel shows real-time memory usage and operation timings
  • Use templates in the Import Wizard for recurring imports with the same structure
  • Check sample data preview in the Import Wizard to verify correct column selection
  • Test with small files first to validate your mapping configuration
